# CSCI 2590: Homework 4 Part 1

> ### Please start early. 

## Set up environment
We recommend you to set up a conda environment for packages used in this homework.
```bash
conda create -n nlp_hw4 python=3.11
conda activate nlp_hw4
pip install -r requirements.txt
```

After this, you will need to install certain packages in nltk.
```bash
python3
>>> import nltk
>>> nltk.download('wordnet')
>>> nltk.download('punkt')
>>> exit()
```
